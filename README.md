# Project Start-Up

1. **Ensure Docker is running.**

2. Navigate to the root directory:
   ```plaintext
   C:\...\CodeTest

3. Open a terminal (CMD o PowerShell) and run the following command:
    ```bash
    docker-compose up --build

4. Open in your browser:
   ```plaintext
   http://localhost:8080

   after the build finishes.
